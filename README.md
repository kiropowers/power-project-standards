## Available Commands

### Knowledge Extraction
- `/extract-from-chat` - Extract from current conversation
- `/extract-by-topic` - Extract by specific topic
- `/extract-from-code` - Analyze code patterns
- `/weekly-summary` - Periodic consolidation

### Foundation Documents
- `/generate-product` - Product overview
- `/generate-tech` - Tech stack docs
- `/generate-structure` - Project structure

### Standards (16 types)
- `/generate-code-review` - Code review checklist
- `/generate-naming-conventions` - Naming rules
- `/generate-error-handling` - Error handling
- `/generate-security` - Security standards
- `/generate-git-workflow` - Git workflow
- `/generate-api-standards` - API design
- `/generate-db-standards` - Database standards
- `/generate-test-standards` - Testing standards
- `/generate-logging-standards` - Logging standards
- `/generate-documentation` - Documentation standards
- `/generate-performance` - Performance guide
- `/generate-async-patterns` - Async patterns
- `/generate-deployment` - Deployment standards
- `/generate-monitoring` - Monitoring standards
- `/generate-dependency` - Dependency management
- `/generate-versioning` - Version control

## Usage

Just use any slash command:

```
/generate-product
```

Kiro will guide you through the process and generate a steering file for your project.
